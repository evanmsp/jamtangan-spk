Aplikasi Sistem Pendukung Keputusan jam tangan. Membantu user memilih jam tangan sesuai dengan kriteria.

Fitur:
1. Memilih jenis jam tangan (Classic Watch, Digital Watch, Smart Watch, Recommendation)
2. Menentukkan kriteria spesifik jam tangan
3. Shop Jam Tangan sesuai jenis jam tangan

Cara Running
1. Buka File di VS Code
2. Klik kanan di index.html
3. Klik Open with live server
